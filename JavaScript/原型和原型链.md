## 基本概念

### 原型 (Prototype)

每个 JavaScript 对象都有一个内部属性 `[[Prototype]]`，指向另一个对象或 `null`，这个被指向的对象就是原型。

```javascript
const obj = { name: 'test' };
const proto = Object.getPrototypeOf(obj); // 获取 obj 的原型
```

### 原型链 (Prototype Chain)

当访问一个对象的属性时，如果对象本身没有该属性，JavaScript 会沿着原型链向上查找，直到找到该属性或到达原型链末端 (`null`)。

```javascript
function Person(name) {
  this.name = name;
}

Person.prototype.sayHello = function() {
  console.log(`Hello, I'm ${this.name}`);
};

const p = new Person('Alice');
p.sayHello(); // 沿着原型链找到 sayHello 方法
```

## 关键属性

### `__proto__`

- 对象的原型指针，指向构造函数的 `prototype`
- 已废弃，建议使用 `Object.getPrototypeOf()` 和 `Object.setPrototypeOf()`

### `prototype`

- 函数对象特有的属性
- 用于构建原型链，作为通过 `new` 创建的实例的原型

### `constructor`

- 原型对象上的属性，指向构造函数本身
- `obj.constructor === Person`

## 原型链示例

```javascript
// 创建对象
const obj = { a: 1 };

// 原型链结构
// obj --> Object.prototype --> null

// 继承示例
function Animal(name) {
  this.name = name;
}

Animal.prototype.eat = function() {
  console.log('eating');
};

function Dog(name, breed) {
  Animal.call(this, name);
  this.breed = breed;
}

// 建立原型链
Dog.prototype = Object.create(Animal.prototype);
Dog.prototype.constructor = Dog;

Dog.prototype.bark = function() {
  console.log('barking');
};

const dog = new Dog('Buddy', 'Golden');
dog.bark();   // 自身方法
dog.eat();    // 原型链上的方法
```

## 相关 API

```javascript
// 获取原型
Object.getPrototypeOf(obj);
obj.__proto__; // 不推荐

// 设置原型
Object.setPrototypeOf(obj, proto);

// 检查原型链
obj instanceof Constructor;
Constructor.prototype.isPrototypeOf(obj);

// 创建指定原型的对象
Object.create(proto);
```

## 注意事项

1. **原型链查找性能**：访问原型链深处的属性比访问自身属性慢
2. **属性遮蔽**：实例属性会遮蔽原型链上的同名属性
3. **`for...in` 遍历**：会遍历原型链上的可枚举属性
4. **`hasOwnProperty`**：区分自身属性和继承属性

```javascript
const obj = Object.create({ inherited: true });
obj.own = true;

for (let key in obj) {
  console.log(key); // inherited, own
}

obj.hasOwnProperty('own');      // true
obj.hasOwnProperty('inherited'); // false
```
