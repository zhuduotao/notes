## 优先级规则

CSS 优先级用于确定当多个规则应用于同一元素时，哪个规则生效。

### 优先级层级（从高到低）

1. **`!important`** - 最高优先级，覆盖所有其他规则
2. **内联样式** - 直接在 HTML 元素上使用 `style` 属性
3. **ID 选择器** - `#id`
4. **类选择器、属性选择器、伪类** - `.class`、`[attr]`、`:hover`
5. **元素选择器、伪元素** - `div`、`::before`
6. **通配符选择器** - `*`

### 优先级计算方式

优先级可以用四元组 `(a, b, c, d)` 表示：

| 选择器类型 | a | b | c | d |
|-----------|---|---|---|---|
| 内联样式 | 1 | 0 | 0 | 0 |
| ID 选择器 | 0 | 1 | 0 | 0 |
| 类/属性/伪类 | 0 | 0 | 1 | 0 |
| 元素/伪元素 | 0 | 0 | 0 | 1 |
| 通配符 | 0 | 0 | 0 | 0 |

### 示例

```css
/* 优先级 (0, 0, 0, 1) */
div { color: red; }

/* 优先级 (0, 0, 1, 1) */
div.class { color: blue; }

/* 优先级 (0, 1, 0, 0) */
#id { color: green; }

/* 优先级 (0, 1, 0, 1) */
#id div { color: yellow; }

/* 优先级 (0, 0, 2, 0) - 两个类选择器 */
.class1.class2 { color: purple; }
```

## 层叠顺序

当优先级相同时，应用以下规则：

1. **后定义的规则优先** - 后面的样式覆盖前面的
2. **来源顺序** - 用户 `!important` > 作者 `!important` > 作者普通 > 用户普通 > 浏览器默认
3. **特异性相同** - 后声明的覆盖先声明的

## 继承

- 某些属性会自动继承（如 `color`、`font-family`）
- 继承的值优先级低于任何选择器匹配的值
- 可以使用 `inherit` 强制继承，`initial` 使用默认值，`unset` 根据属性类型决定

## 注意事项

- `!important` 应谨慎使用，会导致样式难以维护
- 选择器不是越长优先级越高，而是根据选择器类型计算
- 组合选择器的优先级是各部分优先级的总和
